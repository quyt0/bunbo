Welcome to **Bún Bò Huế Tycoon** - an all-in-one static web application that allows you to convert your fiat financial assets into the most powerful and delicious currency in the universe: **Bowls of Bún Bò** (Vietnamese Spicy Beef Noodle Soup).

Instead of stressing over stock markets or dollar exchange rates, let this tool show you exactly how many consecutive days you can survive purely on Bún Bò! 🥢

## ✨ Key Features

- ⚡ **Real-time Auto-Convert:** No submit buttons required! Just type your cash amount and watch the noodle count update instantly. As smooth as sipping hot broth.
- 🎨 **"Delicious" UI/UX:** Designed with a strictly culinary color palette: Chili Sate Red, Broth Orange, Onion Green, and Noodle White.
- 💰 **Flexible Denominations:** Easily select between Thousands, Millions, or Billions of VND.
- 🥩 **Dynamic Pricing:** Built-in standard street prices (30K - 50K VND) plus a custom price input for those high-end, top-tier luxury bowls.
- 🎉 **Tycoon Confetti:** Automatically triggers a massive confetti explosion if your net worth translates to more than 100 bowls! 

## 🚀 How to Use

Since this is a 100% Client-side static web app, there is absolutely zero environment setup required.

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/quyt0/bunbo.git
   ```
2. Open the downloaded folder.
3. Double-click the index.html file to open it in any modern web browser (Chrome, Edge, Safari, Brave...).
4. Input your money and enjoy!

Or check out the Live Demo [here](https://bunbo.izcuyt.com)

## 🛠️ Tech Stack
HTML/CSS: Core structure and interface (Responsive design, CSS Animations).

Vanilla JavaScript: Real-time DOM manipulation and core mathematical logic.

Canvas Confetti: The magical library powering the epic celebratory explosions.
